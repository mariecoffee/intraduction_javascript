---------var---

on créé une variable var
on lui donne un nom ici name (var name)
puis on lui donne une valeur var name = "clément";

ici la valeur "clément" est une string, c'est a dire une chaine de caractère.
une suite de caractère. 

un autre type de variable qui existe c'est les nombres, Numbers.
nombre entier  	// nombre décimaux
var x = 12	//  var x = 25.63

les nombres ne sont pas des string, ils s'afficheront en bleu dans la console 
alors que les strings s'afficheront en noir.

Mais on peut faire passer des nombres en string avec
1) var x = 12
console.log (x); ------ici 12 est en bleu---

2) var xToString = x.toString();
console.log (xToString);  ---- ici 12 est en noir car on a passé le nombre en string



une string sera tjs ecrite en guillement, soit double guillement soit le simple guillement.

les strings ont une propriété bcp utilisés, c'est la propriété lenght

par exemple:

var name = "clement";

var nameLength = name.length

console.log (nameLength);

résultat affiché est 7

-----------plusieurs fonctions utilisés avec string---
si par exemple dans la string 

var myString = "salut Clément";

Je veux savoir a quelle position de trouve "Clément",

je fais:  var position = myString.indexOf ("Clément");

console.log (position)
cela affichera 6

!!!!!!!!!!!!!!!!!
dans ce nouvel exemple, on veut remplacer le mot Clément par le mot Flavie

exemple:

Var myString = "Bienvenue Clément";

console.log (myNewString);
